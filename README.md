# Marketing Funnel & Conversion Performance Analysis

## Project Overview

This project analyzes marketing campaign data to evaluate customer conversion performance and identify key drop-off points in the marketing funnel. The objective is to understand how leads move through the conversion process and provide actionable recommendations to improve business outcomes.

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Dashboard Design

## Dataset

The analysis uses the Bank Marketing Dataset, which contains customer and campaign information from a direct marketing campaign conducted by a banking institution.

## Key Performance Indicators (KPIs)

* Total Contacts: 45,211
* Total Conversions: 5,289
* Conversion Rate: 11.70%
* Drop-off Rate: 88.30%

## Analysis Performed

* Conversion Distribution Analysis
* Conversion by Job Category
* Conversion by Contact Method
* Conversion by Education Level
* Funnel Performance Evaluation
* Drop-off Analysis

## Dashboard Features

* KPI Summary Cards
* Conversion Distribution Pie Chart
* Job Category Conversion Analysis
* Contact Method Performance Analysis
* Education Level Conversion Analysis

## Key Insights

* The overall conversion rate was 11.70%.
* Management and Technician segments generated the highest number of conversions.
* A significant percentage of contacts did not convert, indicating funnel drop-off opportunities.
* Conversion performance varied across contact methods and customer demographics.

## Recommendations

* Focus marketing efforts on high-converting customer segments.
* Improve lead nurturing and follow-up strategies.
* Optimize underperforming contact channels.
* Implement targeted campaigns based on customer characteristics.
* Continuously monitor conversion metrics to improve campaign effectiveness.
